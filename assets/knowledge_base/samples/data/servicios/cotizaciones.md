# Cotizaciones de proyectos
Medif Estructuras cotiza proyectos completos (estructurales, hidrosanitarios, puentes grúa) con énfasis en análisis técnico y entregables listos para aprobación.

## Intención: proceso de cotización
- Enviar planos completos (arquitectura, estructura, instalaciones o hidrosanitarios), normativas, cargas y plazos al correo `eduardo.mediavilla@medifestructuras.com`.
- Solicitar el link de pago Stripe en la misma comunicación o indicar “Enviar mis datos” para recibirlo.
- Resultado: el equipo responde con preguntas de validación, un alcance técnico detallado y el presupuesto final.

### Palabras clave
- cotización de proyectos, plan de cotización, link Stripe, planos y normativas, contacto Eduardo.

## Proceso recomendado
1. Revisa los cursos y servicios para entender el alcance, luego prepara la documentación del proyecto.
2. Envía a `eduardo.mediavilla@medifestructuras.com` los planos (PDF/DWG), ubicación, tipo de estructura, usos y niveles.
3. Incluye: normativas aplicables, cargas especiales, restricciones de plazos, referencias de proyecto o fotos.
4. El equipo responde con preguntas adicionales, alcance del cálculo, plazos y estimación económica.
5. Cuando el alcance se valida, confirmamos honorarios definitivos y formas de pago.

## Información adicional que ayudamos a definir
- Precios referenciales por tipo: **0,5 a 3 USD / m²** para proyectos estructurales típicos (diseño ejecutivo). Para estudios especiales (análisis no lineal, puentes, naves industriales) se detalla un alcance personalizado.
- Se entregan memorias, planos, especificaciones, hojas de cálculo y modelos BIM.
- Cada cotización se documenta con la lista de requisitos, entregables y entregas parciales.

## Políticas de contacto
- `eduardo.mediavilla@medifestructuras.com` es el único canal para cotizar diseño estructural o hidrosanitario.
- No se redirige a correos alternos para este tipo de solicitud.
- Una vez recibimos los planos, confirmamos en menos de 24 horas qué información falta para cerrar la cotización.

- Ofrecemos plantillas para adjuntar información básica (ubicación, posibles interferencias, prioridades).
- Los proyectos grandes se planifican con revisiones técnicas y el acompañamiento del foro para resolver dudas específicas.
